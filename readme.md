# Quantum Distribution Loading Benchmarking

Implementation of systematic benchmarking framework for variational quantum circuits loading probability distributions, following Dasgupta and Paine (2022) methodology.

## Setup

```bash
# Create environment
conda create -n qdist-bench python=3.9
conda activate qdist-bench

# Install dependencies
pip install -r requirements.txt

# Verify setup
python scripts/verify_setup.py
```
